# Visily Code Export

This project was generated by [Visily](https://www.visily.ai/). Follow these steps to view your design in the browser.

## Prerequisites

Ensure you have [Node.js](https://nodejs.org/en/download) (version 22 or later) installed on your machine.

## How to Run

1. Open your terminal and navigate to this folder.
2. Start a static file server:
   ```bash
   npx serve .
   ```
3. **View your design!** Once the server starts, open your browser and go to the link shown in your terminal (usually http://localhost:3000).

Alternatively, you can double-click `index.html` to open it directly.
